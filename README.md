# 🛡️ SOC Monitoring Dashboard

A Full Stack SOC (Security Operations Center) Monitoring Dashboard built using React, FastAPI and Recharts.

## Features

* Real-time Threat Monitoring
* Security Alert Dashboard
* Threat Severity Cards
* Live Clock
* Alert Counter
* Threat Distribution Chart
* FastAPI Backend APIs
* React Frontend
* Responsive Dark Theme UI

## Tech Stack

### Frontend

* React
* Vite
* Recharts
* CSS

### Backend

* FastAPI
* Python

## API Endpoints

### Threat Summary

`/threats`

Returns threat severity statistics.

### Security Alerts

`/alerts`

Returns recent security alerts.
